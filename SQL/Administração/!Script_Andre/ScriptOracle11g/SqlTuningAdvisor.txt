+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
To disable the automatic SQL tuning advisor task, just replace the ENABLE procedure 
with the DISABLE procedure, as shown here:
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
begin
dbms_auto_task_admin.disable (
client_name => 'sql tuning advisor',
operation => 'NULL',
window_name='NULL');
end;

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
You can enable the automatic SQL tuning advisor task using the ENABLE procedure, as shown here:
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

begin
dbms_auto_task_admin.enable (
client_name => 'sql tuning advisor',
operation => 'NULL',
window_name='NULL');
end;


+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
Shows how to configure an automatic SQL tuning task to automatically accept 
SQL profiles recommendations made by the SQL Tuning Advisor:
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

begin
dbms_sqltune.set_tuning_task_parameters(
task_name => 'SYS_AUTO_SQL_TUNING_PROG',
 parameter => 'accept_sql_profiles', value => 'true');
end;
