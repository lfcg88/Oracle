Desabilitar Partitioning - DWDES

1- Baixar o banco

2- Entar no diretório $ORACLE_HOME/bin

3- Executar o comando abaixo

 chopt disable partitioning



Desabilitar Oracle Tuning Pack / Oracle Diagnostic Pack / 

1- Executar o comando abaixo

alter system set control_management_pack_access=none scope=both;
